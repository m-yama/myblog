Zenwatch3をAndroidに接続したらバッテリーが長持ちした¶

Zenwatch3をAndroidケータイに接続して使用すると、iPhoneで使用した場合に比べて、バッテリー持続時間が大幅に向上した。

バッテリー残量に関する記録¶

先日の、 Zenwatch3のバッテリー持続時間が短い件 では、iPhoneに接続して使用した場合にバッテリーがあまり持たないという結果だったが、Androidケータイ(docomo SC-01F Galaxy Note 3) に接続して使用すると、かなり長持ちした。

所感¶
